Summary

  • Rothemberg’s counting is only sufficient

  • Paper provides necessary conditions

  • Observational equivalence: For Gaussian SVAR, implies that two structural paramter points have the same reduced form representation. They are observationally the same.

  • They show that if two structural paramter points are observationally equivalent, then the one point is a rotation of the other point.

  • Global identification: No other observationally equivalent parameter point exists

  • Local identification: No other observationally equivalent parameter point exists in a neighbourhood

  • We are more interested in global identification

  • partial identification: jth equation is globally identified if and only if there exists no observationlly equivalent paramter point whose jth equation is different; there exists no point \((\tilde A_0, \tilde A_+)\) such that \(A_0e_j \neq \tilde A_0e_j\) or \(A_+e_j \neq \tilde A_+e_j\).

  • So when is the jth shock globally identified? jth shock is identified whenever jth equation is identified.

Theorem 7: Necessary and Sufficient Condition for Global Identification

In words: SVAR is exactly identified if and only if the last shock has zero restriction, the second to last has one restriction, etc. when the shocks are ordered in such a way that the last has the fewest restrictions, the second to last has the second to fewest restritions etc. Conditions on restrictions are satisfied when putting restrictions on IRFs or structural matrices.

Statement: Consider and sVAR with admissible and strongly regular restrictions represented by \(R\). The SVAR is exactly identified if and only if \(q_j = n-j\) for \(1\leq j \leq n\)

  • What does exact identification mean?

  • SVAR is exactly identified if and only if there exists a unique structural paramter point satisfying the restrictions for almost all reduced form paramter points.

  • Theorem 7 tells us when SVAR is exactly identified. Would also like to obtain this exactly identified structural parameter point. Theorem 5 gives a way to find such a point

Important

In words: Theorem tells us that if the SVAR is exactly identified, then the exactly identified structural paramter point is a rotation of some structural paramter point that does not satisfy all restrictions.

Statement: Consider an SVAR with restrictions represented by \(R\). The sVAR is exactly identified if and only if, for almost every structural paramter point \((A_0, A_+)\in U\), there exists a unique matrix \(P\in O(n)\) such that \((A_0P, A_+P)\in R\)

  • can combine Theorem 5 and 7 to find exactly identified point.
  1. Obtain some structural paramter point in \(U\), i.e. the point found via a Cholesky identification.
  2. Find rotation matrix \(P\in O(n)\) such that \((A_0P, A_+P)\in R\)
